# Tap Stack

Tap Stack is a fast-paced arcade stacking game where players tap to drop blocks and build the tallest tower possible.

The game focuses on timing, precision, and score chasing mechanics designed for quick mobile play sessions.

## Features

• Simple one-tap gameplay  
• Timing-based block stacking  
• Increasing difficulty as the tower grows  
• Score-based gameplay loop  

## Development

This project was developed using **Base44**, an AI-powered no-code development platform.

The development workflow involved:

• AI-assisted feature prototyping  
• Prompt engineering for gameplay logic  
• Iterative testing and refinement
